依赖注入
程序的耦合
什么是程序的耦合：
	 程序间的依赖关系
耦合包括：
	 类之间的依赖
	 方法之间的依赖
	 
耦合性高会造成程序的独立性差

什么是解耦：
	降低程序间的依赖关系
	在编译器不依赖，运行时才依赖
思路：
	1.使用反射来创建对象，而并非使用新的关键字
	2.通过读取配置文件来获取要创建的对象全限定名
